# IOT_07_WatchDog
Desafio desenvolvido com ESP32 como prática do componente, usando de bibliotecas para conexão Wifi e teste de requisições HTTP, verificando a existência da rede e validação de resposta do servidor (Sinalizando pelo Led). Desenvolvido em aula da de IOT do curso Desenvolvimento de Sistemas do Senai

Segue um vídeo do código sendo executado em circuito:
https://youtu.be/Wau44QjFRl8?si=z5PcvkTtlLt9VM0G

## Tecnologias utilizadas
* C++
* Arduino IDE
* Wokwi
* ESP32
